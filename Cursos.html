Desenvolvimento Java - Iniciante

O Curso

O curso de desenvolvimento java para iniciantes ensina aos jovens programadores como funciona a linguagem, como podemos utiliza-la e também a fazer pequenos desenvolvimentos com a linguagem. 

Requisitos Mínimos: 
- Lógica de Programação
- Computador com Intel Core i5 ou similar + 8GB de RAM + HD 500GB
- Força de vontade

Em cada etapa do curso, o aluno deverá enviar ao professor um feedback da aula, informando o que aprendeu e suas dúvidas. Após assistir todas as aulas e enviar todos os relatórios ao professor o aluno recebera o certificado de conclusão da Consultoria CJL, válido em todo território nacional.
